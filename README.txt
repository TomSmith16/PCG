Be sure to delete all of the clones and region game objects from the scene, as this is where the procedural content is generated.

I had trouble removing all the content when trying to generate a new level, so generating while the content is still in the level will create a large amount of assets and likely crash Unity3D.
*************************************************************************************************************

When you first open the Unity level, the map will be white, this seems to be an issue with the map generator, although it can be solved by generating a new map in the MapGenerator game object.
